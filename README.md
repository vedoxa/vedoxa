# 📚 VEDOXA - Premium Digital Library

Welcome to the official repository of **VEDOXA** — A premium platform designed to awaken your consciousness with 100% original, verified digital books on spirituality & psychology.

🌐 **Live Website:** [vedoxa.shop](https://vedoxa.shop)

---

## ✨ Key Features
- **Instant Secure Access:** Zero-wait time. Books are unlocked instantly upon successful payment.
- **Premium Web Reader:** A highly secure, anti-piracy embedded PDF reader for a seamless reading experience. No zooming or downloading allowed to protect content integrity.
- **Automated Help & Support:** An intelligent, floating bot interface to handle customer FAQs and automated ticketing system.
- **Native Web Share:** Integrated sharing API for smooth link distribution across platforms like WhatsApp, Instagram, and Telegram.
- **Dynamic Reward System:** Earn points on every purchase and redeem them instantly on the next checkout.

---

## 🛠️ Tech Stack
This project is built using modern, highly scalable technologies:
- **Frontend:** Next.js (React), Tailwind CSS, Framer Motion (for smooth animations)
- **Backend & Database:** Supabase (PostgreSQL, Authentication, Storage)
- **Payment Gateway:** Razorpay (Seamless & Secure INR transactions)
- **Icons & UI Elements:** Lucide React

---

## 🚀 Performance & Security
- Fully responsive and optimized for mobile devices.
- Prevented native touch-zooming to ensure app-like UI consistency.
- Environment variables and Row Level Security (RLS) implemented via Supabase for maximum data protection.
